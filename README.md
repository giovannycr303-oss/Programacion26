Sistema de Gestión de Tickets (Help Desk)La aplicación es un Single Page Application (SPA) de administración de incidentes de TI, desarrollada para la persistencia local de datos y la gestión de flujos de trabajo.
Arquitectura de Datos: Implementa un patrón de CRUD (Create, Read, Update, Delete) completo mediante la manipulación de objetos en memoria y persistencia persistente utilizando la API localStorage (soportesTiTickets).  Gestión de Estado y Lógica: El sistema utiliza la manipulación directa del DOM para la renderización dinámica de tablas y métricas. La lógica incluye el filtrado de datos en tiempo real (por estado o búsqueda de texto) y la serialización para exportación de archivos (JSON/CSV).  Interfaz y Estilo: Desarrollada con un sistema de diseño modular basado en CSS Variables para la gestión de temas, empleando Flexbox y Grid para un layout responsivo y adaptativo.  Funcionalidades Técnicas:Validación de Formularios: Validación de campos obligatorios y formato de correo electrónico.  Dashboard: Cálculo dinámico de métricas (totales, pendientes, resueltos) mediante la iteración de arreglos.  Interactividad: Uso de un modal para la visualización detallada de registros y botones de acción para transiciones de estado
